## 📝 使用方法

### 1. 预览文件
```bash
open ankiweb-description.html  # macOS
```
或直接在浏览器中打开该文件。

### 2. 复制到 AnkiWeb

1. 打开 ankiweb-description.html
2. 全选并复制所有 HTML 内容（Cmd+A, Cmd+C）
3. 登录 https://ankiweb.net/
4. 找到您的卡牌（ID: 1584745637）
5. 点击 Edit
6. 粘贴到 Description 字段
7. 点击 Save

### 3. 检查效果

保存后访问卡牌页面，确认：
- ✅ 排版正确
- ✅ 链接可点击
- ✅ 表格显示正常
- ✅ Emoji 正常显示
- ✅ 引用框格式正确
