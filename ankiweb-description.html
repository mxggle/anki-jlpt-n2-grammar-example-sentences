JLPT N2 新完全掌握语法 – 例句集（带音频、例句解析、接续高亮）。面向 JLPT N2 高频语法巩固与输出练习。每张卡片配有自然语速音频、例句逐行解析与「接续形式」高亮，帮助在理解与听辨中同时掌握语法用法与搭配。若发现错漏或希望添加语法点，欢迎反馈。

⚠️ 本语法卡组基于《新完全掌握 JLPT N2 语法》练习书制作。本卡组仅作为该书的辅助学习资料，因此缺少了书中大量的原始内容。如果你计划长期使用本卡组，请支持正版，购买原书。

Grammar deck for the Shin Kanzen Master N2 JLPT practice book. This deck is meant to be used in combination with the book and is therefore missing much of the original content. Please support the publisher by purchasing the book if you plan to use this deck.

💡 学习建议

  1. 按课程顺序学习 - 卡牌按照教材的课程顺序组织（Lesson 01开始）
  2. 注意语法接续 - 每个语法点都标注了具体的接续方式
  3. 利用音频 - 充分利用例句音频练习听力和发音
  4. 理解逐词精解 - 仔细阅读逐词精解部分，理解句子结构
  5. 定期复习 - 利用Anki的间隔重复算法定期复习
  6. 配合教材 - 建议配合《新完全掌握N2语法》教材使用效果更佳

📥 使用方法

  1. 点击页面上的"Download"按钮下载 .apkg 文件
  2. 双击下载的文件，Anki会自动打开并导入
  3. 开始学习！

注意：首次导入时，音频文件会自动同步到Anki媒体库，可能需要一些时间。

❓ 常见问题

Q: 音频为什么不播放？

A: 请确保已完成媒体文件同步。在Anki菜单中选择"工具" → "检查媒体库"。

Q: 可以在手机上使用吗？

A: 可以！支持AnkiMobile（iOS）和AnkiDroid（Android）。需要通过AnkiWeb同步。

Q: 如何按课程筛选学习？

A: 使用Anki的标签功能，或在浏览器中按"Lesson"字段筛选。

Q: 更新会覆盖我的学习进度吗？

A: 不会。按照上述"更新说明"操作，学习进度会完整保留。
